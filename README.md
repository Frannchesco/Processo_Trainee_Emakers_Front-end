# Mindemy – Plataforma de Cursos Online

Mindemy é uma plataforma de cursos online desenvolvida com **React + Vite**, oferecendo uma experiência moderna e responsiva para alunos que desejam adquirir habilidades práticas em tecnologia, design e gestão.

Este projeto foi desenvolvido como trabalho prático da disciplina **Empreendedorismo em Sistemas de Informação** da UFLA, e atende aos requisitos de front-end com componentes reutilizáveis, roteamento, consumo de assets e design customizado.
